# TravelMate
This application works as a Multiagent AI LangGraph travel planner. You need to mention a travel location, and this will give you the entire plan with respect to the flights, hotels, budget, and daily travel itinerary. This is a useful application for people interested in traveling to a different country; it will help them make an estimated plan.

## How to run?
1. Create the Virtual Environment

```bash
conda create -n travel python=3.11 -y
```

2. Activate the Environment

```bash
conda activate travel
```

3. Install the requirements

```bash
pip install -r requirements.txt
```
